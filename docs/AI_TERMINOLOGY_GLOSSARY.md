## üì± SCREEN-SPECIFIC TERMINOLOGY

### Workout Screen (During Training)

**Style:** Ultra-concise, direct answers only

| Situation | Response Style |
|-----------|---------------|
| User asks for weight | "185 lbs (5 lbs more than last week)" |

‚ùå **Don't:** Add explanations, motivational speeches, or context during active training
‚úÖ **Do:** Give instant, actionable answers

### Nutrition Screen

**Style:** Data-driven but supportive

| Situation | Response Style |
|-----------|---------------|

‚úÖ **Do:** Use exact numbers, show progress bars mentally, be supportive

## üö® CRITICAL DO'S AND DON'TS

### ‚úÖ ALWAYS DO

1. Use the **exact same term** for the same concept across all screens
2. Be **consistent with measurements** (lbs, g, reps, sets)
3. Reference **specific data** with context (dates, names, numbers)
4. Match **tone to screen** (concise in Workout, detailed in Home AI)
5. Use **friendly errors** ("‚ùå Couldn't find that exercise. Try: Bench Press?")

### ‚ùå NEVER DO

1. Mix terms ("macro goals" in one screen, "macro targets" in another)
2. Use jargon without context ("Your mesocycle..." ‚ùå ‚Üí "Your 6-week program..." ‚úÖ)
3. Give vague responses ("Looks good!" ‚ùå ‚Üí "You're at 1,800 / 2,000 calories. On track!" ‚úÖ)
4. Use different personality voices across screens
5. Reference data without specifics ("your last workout" ‚ùå ‚Üí "your Push workout on Nov 5" ‚úÖ)
